# 🌍 Gapminder Animated Data Visualization

An interactive animated bubble chart built using **Python** and **Plotly Express**.

The project visualizes the relationship between **GDP per Capita**, **Life Expectancy**, **Population**, and **Continents** over time using the famous **Gapminder Dataset**.

---

## ✨ Features

- 🌍 Animated Bubble Chart
- 📈 GDP per Capita vs Life Expectancy
- 👥 Bubble Size represents Population
- 🎨 Color-coded Continents
- 📅 Year-wise Animation
- 🌙 Dark Theme
- 🖱 Interactive Hover Information
- 📊 Logarithmic X-Axis
- 💾 Export to HTML

---

## 📷 Preview

The visualization displays:

- GDP per Capita on the X-axis
- Life Expectancy on the Y-axis
- Bubble Size based on Population
- Bubble Colors based on Continents
- Smooth Year Animation (1952–2007)

## 📦 Installation

Install dependencies

```bash
pip install plotly
```

Run the project

```bash
python app.py
```

---

## 📁 Output

The project generates

```
animated_scatter_plot.html
```

Open the HTML file in any browser to explore the interactive visualization.
